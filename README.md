# Cheat Stengine

Cheat Stengine is a tool should be able to achieve everything that Cheat Engine can do, but with a focus on simplicity, ease of use, and modern design principles.
It is built from scratch in C++ to ensure good performance and maintainability.

> [!WARNING]
> Cheat Stengine is still in the very early stages of development. Many features are not yet implemented, and the tool may be unstable.

## Compiling from Source

```console
cmake -S . -B build
cmake --build build --config Release
```
